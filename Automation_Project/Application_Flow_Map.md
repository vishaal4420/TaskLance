# TaskLance Application Flow Map

## 1. Authentication Flow
- Splash Screen -> Onboarding -> Login/Signup
- Login -> (Role Check) -> Dashboard
- Signup -> Email Verification -> Profile Setup -> Dashboard
- Forgot Password -> Reset Link

## 2. Freelancer Workflow
- Dashboard -> Find Work -> Project List -> Project Detail
- Project Detail -> Proposals -> Create Proposal
- Project Detail -> Tasks -> Kanban Board -> Task Detail -> Time Tracker
- Project Detail -> Milestones -> Milestone List -> Milestone Detail
- Project Detail -> Deliverables -> Upload Deliverable
- Invoices -> Create Invoice
- Expenses -> Add Expense

## 3. Client Workflow
- Client Dashboard -> Create Project
- Project Detail -> Proposals -> Review Proposal -> Accept/Reject
- Project Detail -> Milestones -> Approve Milestone
- Project Detail -> Invoices -> Pay Invoice
- Client Portal -> View Project Progress

## 4. Communication
- Inbox -> Chat Room -> Text/Media/Video Call
- Support -> FAQ / Support Chat

## 5. Profile & Settings
- Edit Profile
- Billing & Payments
- Team Management
- Security: Change Password, 2FA
- Legal Documents
